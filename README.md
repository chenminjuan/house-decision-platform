# 首套房决策助手

> **"贝壳帮你找房，我们帮你做决定。"**

面向中国三四线城市首次购房者的决策辅助工具。基于 10.6 万+ 真实二手房源数据，帮助用户回答三个核心问题：

1. **我真正买得起多少钱的房子？**（不是总价，是落地成本）
2. **哪些房源最适合我？**（不是筛选，是个性化推荐）
3. **我该怎么做决定？**（不是浏览，是可执行的决策报告）

---

## 🎯 产品定位

| 维度 | 市场现有产品 | 本产品 |
|------|:---:|:---:|
| 核心模式 | 信息撮合 / 广告展示 | **决策辅助** |
| 目标用户 | 已有目标的搜索型用户 | **"不知道买什么"的首次购房者** |
| 目标市场 | 一二线城市为主 | **三四线城市（下沉市场）** |
| 价格视角 | 挂牌总价 | **落地总成本（含税费+中介费+装修）** |
| 推荐逻辑 | 条件筛选 + 默认排序 | **多维度加权评分（6维+用户自定义权重）** |
| 输出物 | 房源列表 | **结构化决策报告 + PDF导出** |

---

## 📊 数据资产

- **数据来源**：房天下（fang.com）二手房挂牌数据
- **数据规模**：106,118 条房源记录
- **覆盖范围**：513 个城市，30+ 个省份
- **数据字段**：城市、省份、小区名称、地址、价格、面积、户型、朝向、楼层、建成年份、单价
